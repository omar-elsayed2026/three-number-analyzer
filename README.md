**Number Analyzer** is a simple yet practical Python application built with **Tkinter** that provides an easy-to-use graphical interface for analyzing three numbers.

Users can enter three integer values and instantly:

* 🔹 Find the **largest number**
* 🔹 Find the **smallest number**
* 🔹 Calculate the **total sum**

This project is designed for beginners who are learning Python and GUI development. It demonstrates the use of **Tkinter widgets**, **functions**, **button commands**, and Python's built-in `max()` and `min()` functions in a clean and straightforward way.

### ✨ Features

* 🖥️ Simple graphical user interface (GUI)
* 🔢 Analyze three user-entered numbers
* 📈 Find the maximum value
* 📉 Find the minimum value
* ➕ Calculate the total sum
* ⚡ Fast and lightweight
* 🐍 Built entirely with Python and Tkinter

This project is a great starting point for beginners who want to practice creating desktop applications using Python while improving their programming and problem-solving skills.
